# 刹那の見斬り

バニラ JavaScript + HTML5 Canvas で作られた反射神経ゲームです。スマホの縦画面ブラウザで動作します。

## ファイル構成

```
index.html   … エントリーポイント
style.css    … レイアウト・タッチ無効化スタイル
game.js      … ゲームロジック全体
assets/      … 差し替え用画像を配置するフォルダ
```

## 画像アセット

`assets/` フォルダに以下の PNG 画像を配置することで、フォールバック矩形から実際のグラフィックに差し替えられます。

| ファイル名            | 用途                     |
|-----------------------|--------------------------|
| player_idle.png       | プレイヤー 待機          |
| player_attack.png     | プレイヤー 攻撃          |
| player_dead.png       | プレイヤー やられ        |
| enemy_idle.png        | 敵 待機                  |
| enemy_attack.png      | 敵 攻撃                  |
| enemy_dead.png        | 敵 やられ                |
| bg_image.png          | 背景                     |
| slash_effect.png      | 斬撃エフェクト           |

画像が存在しない場合はカラー矩形でフォールバックされるため、画像なしでもゲームは動作します。

## ゲームの流れ

1. **START** – タップで開始
2. **WAITING** – 「かまえて！」が表示される（2〜5秒）。この間にタップすると **フライング** で敗北。
3. **SIGNAL** – 「！」が表示されたらすぐタップ。CPU の反応時間はランダム（200〜500ms）。
4. **RESULT** – 勝敗と反応時間を表示。タップで最初に戻る。
